## 📌 Next-Generation Code Agents @ AAAI 2026

Website: [https://sites.google.com/view/next-gen-code-agents-aaai26/home](https://sites.google.com/view/next-gen-code-agents-aaai26/home)

This workshop brings together researchers and practitioners to explore the coming wave of intelligent coding agents — from human–AI collaboration and self-repairing code to testing, trust, and scalable deployment. The schedule includes invited talks from industry leaders, paper presentations, and a closing panel with top academic and industrial experts.
